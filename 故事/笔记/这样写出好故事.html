# 情节到底是什么

LOCK

# L主角lead

  • 稳固的情节永远始于有趣的主角。最好的情节中，主角必须引人注意，迫使我们从头到尾盯着他
  • 活用车祸现场，如同开车看到车祸现场，人会停下来张望。读者也喜欢看复杂的角色犯下无可挽回的错，把人生搞得一塌糊涂。

# O目标objective

目标是小说的驱动力

- 想取得某样事物
- 想逃离某样事物

扎实的情节中，主角只有一个主要目标，这个目标会形成故事问题：主角会达成目标么？ 提升主角目标的重要性

1. 死亡威胁
2. 相关性

# C冲突confrontation

来自其他角色或外在环境的组里能让你的故事活过来

# K冲击结尾knockout

没人喜欢平手的拳击比赛，希望看到KO击倒

# 到底有多少种情节

虽然情节有很多变体，但是每一种都符合LOCK系统：主角有强烈追求的目标，却被迫面对冲突，而他的故事贯穿整本小说。

# 结构：稳固你的情节

# 三幕结构

开头、中段(混乱)、结尾

# 开头

开头永远聚焦于故事里的任务，让读者尽快与主角产生共鸣

  • 呈现小说中的世界
  • 奠定小说的调性
  • 说服读者继续读到中段
  • 介绍对手出场

# 中段

主角与对手之间一系列的攻防战。支线增加复杂性。

  • 加深角色之间的关系
  • 让观众持续关心故事发展
  • 铺陈替故事收尾的最后一战

# 结尾

  • 把所有梗收干净
  • 创作余韵

# 扰乱事件及两扇门

# 扰乱事件

一定发生某些事，让我们角色面临威胁或挑战，从而离开原地

# 两扇门

从开头到中段，从中段到结尾。两次转折点，无法折返的门。
如果还能如前一样生活，那就好没有穿过。
第一起扰乱时间跟第一扇无法折返的门并不相同。
穿过第一扇门之后，主角必须探索自己内心，展现勇气，学习新的事物，结交新的盟友等。
想从中段进入结尾，穿越第二道门，通常是一条重要线索或信息，或者重大挫折和危机。

# 如何头脑风暴情节点子

寻找情节点子，永远从自己开始。

# 人格筛选网

  • 世界上你最关心的事是什么
  • 如果自己写讣闻，怎么写
  • 你的外表如何，你对自己长相有何感想，这对你有何影响
  • 你最害怕什么
  • 你最主要的缺点是什么
  • 你的强项是什么，你希望能有什么强项
  • 如果你能做一件事，而且必须成功，你会做什么
  • 童年时期有哪三件事塑造了现在的你
  • 你有哪些惹人讨厌的习惯
  • 你有那些秘密希望永远不要公开
  • 你的人生哲学

# 追寻点子

没有时间浪费在普通的故事上 前置规则

  • 确定点子时间，每周至少一次
  • 呆在安静的地方，让自己放松
  • 给自己三十分钟不受打扰
  • 从以下练习中挑选一项或数项，先读介绍
  • 首先，放手让你的想象力随兴生长，把每个点子都记在纸上或者计算机上
  • 不要给自己设限，任由点子涌现，不要妄下结论
  • 尽情享受想点子的过程，玩的开心
  • 把每个点子记下来
  • 两三次之后开始评估你的点子
  • 尽可能多重复想点子过程

# 想出几百种情节点子的二十种方法

  • 如果游戏
    • 读报纸，每读一篇就问如果呢
    • 每看一个电视或广告，就问如果呢
    • 让大脑自由奔驰
    • 把如果问题列表
    • 把表放一边，过几天回来看，挑几个可以发展的问题。多记笔记
  • 书名
    • 先起一个很酷的书名，然后写出相符的作品
  • 列表
    • 先让脑袋爬梳记忆中的画面，然后快速记下潜意识的名词
  • 议题
    • 哪些议题让你热血沸腾？列出你的议题
  • 亲眼去看
    • 现在这个瞬间，我到底想写什么，列出首先想到的三件事，持续写二十分钟，持续五天，休息一天把感兴趣的挑出来。
  • 亲耳去听
    • 听音乐，看看浮现什么画面
  • 角色优先
    • 想象外表
    • 重塑你认识的人
    • 讣闻
    • 最糟的事 他身上可能发生的最糟的事
  • 向大师取经
    • 独创性就是让你剽窃成功的关键，可以截取前人采用的形式
  • 推翻类型
    • 阅读类型小说，读者会期待特定的节奏和发展，利用他们的期待完成大转弯。
    • 外太空的西部故事
  • 预测趋势
    • 谁会在乎这个主题
    • 那个人明年如何应对这个议题，十年后呢
    • 如果整个社会都接受这个议题，会怎么样
    • 如果整个社会都排斥呢
    • 这项议题最容易伤害到谁
  • 在报纸中挖宝
    • 自问一系列如果问题，充实找到的点子
  • 研究
    • 从你一直有兴趣的领域，挑一本非文学作品
    • 大略翻一遍，了解大意
    • 写下你想到的情节点子
    • 把书详细读完
    • 记下更多的点子，同时发展先前看到的点子
    • 只要这么做，很快就会找到一条信息，让你热血沸腾
  • 其实我真正想写
    • 首先写下 其实我真正想写，然后一口气写十分钟，让脑中的想法扩充联结这些点子
  • 偏执
    • 偏执会控制角色最深沉的情绪，促使他所有行动。因此适合当寻找点子的跳板
    • 创造一个角色，让他为了某件事偏执，再看她发展
  • 开场白
    • 写出开场白，进行联想扩展
  • 写序章
    • 紧凑的序章可以带出很多点子，试写一两千序章练笔
  • 联想图
    • 准备 挑一个或概念发展
    • 出手 从链接中延伸其他联结
    • 瞄准 想到的联结会带来新方向，方向转移
  • 感动人心的结尾
    • 在脑中剧院想象一场高潮结尾
    • 倾听搭配的配乐
    • 让各式各样的情绪涌现
    • 随心所欲加入角色，增加冲突
    • 尝试以不同方式呈现同一主题，直到想出令人难忘的点子，然后问自己
    • 故事中有哪些角色
    • 什么事情让他们聚在一起
    • 我要如何回朔到故事起点 先想好可能的结局是最方便的写作指南。
  • 职业
    • 研究别人的职业能找出很多题材
    • 看新闻，杂志的时候把有趣的工作记下来
  • 绝望
    • 写就对了

# 培养你的点子

  • 挑出你最喜欢的点子，写出他的诱因，标语和弱点

# 贝尔金字塔

  • 作品必须独特，但又不能独特到营销部门不止如何是好
  • 用贝尔金字塔审视点子
  • 热情是基础
  • 出版的潜能，来自于作者独特的文风和远见
  • 精准 不要脱离目标

# 强劲的开头

目标：

  • 吸引读者
  • 建立读者和主角之间的联系
  • 介绍故事中的世界
  • 奠定小说的基本调性
  • 说服读者继续读到中段
  • 介绍出场对手

# 通过主角建立与读者联系

  • 认同 让读者认为主角与自己类似
  • 同情
    • 危机 立即面对危险，他马上变得值得同情
    • 困难 读者欣赏努力克服困难的人
    • 弱者 读者喜欢面对巨大挑战的人
    • 脆弱 感觉主角随时会被击倒
  • 喜爱
    • 赋予不讨喜的角色其他吸引人的特色
  • 内心冲突
    • 将主角的疑虑摊在情节表面，就能将读者进一步拉近故事

# 三种拖累故事的方式

  • 过度描述
  • 回首过去
  • 没有威胁

# 说服读者到中段

  • 先有动作，再做解释
  • 解释的时候模仿冰山，只暴露十分之一
  • 将信息包装在冲突中

# 中段

用场景延续紧张的情绪，提高角色面对的代价，让读者一直担心，并用看似命中注定的方式，将故事推向第三幕

# 死亡

  • 精神上的死亡

麦田里的守望者魅力历久不衰，显然是许多对这都对童年和成年之间短暂的青少年时代有共鸣，而在这个灰色地带，许多人都在寻找活下去的理由

所以主角的目标必须和他的命运息息相关

  • 职业生涯上的死亡

读者担心主角失去什么，他们才可能继续读下去

# 对手

  • 将对手设定为人
  • 如果对手是一群人，从中间挑一个作为领袖
  • 让对手比主角还强

# 附着元素

强烈的关系或者情境，将角色绑在一起

一定要想出理由，解释主角和对手为什么不能停止缠斗。

写下数个冲突场景，大部分都以主角遭到挫败做结，借此强迫他重新分析处境，采取其他方法达到目标。

一来一往，闪避后又进击。 这就是小说的核心。

以下是附着元素更有力的建议

  • 生与死
  • 双方冲突与职责有关
  • 道德义务
  • 偏执
  • 优势故事的地点能将双方困在一起

# 准备好冲突用的武器

行动、反应、继续行动，这是故事的基本韵律

# 吸引读者读下去

  • 拉长紧张段落

  • 提高付出的代价

    • 情节造成的代价

    有时主角面临外来的威胁，这类威胁都是另一角色想伤害主角。不管是实际伤害，还是心理或者职业上的

    • 内心的代价
      • 怎样才能让角色更纠结
      • 主角在乎的人有没有办法陷入麻烦
      • 主角有没有过往见不得人的秘密可以揭露
    • 社会造成的代价

# 如何唤醒昏昏欲睡的中段

  • 分析故事中的代价
  • 加强附着元素
  • 提升情节的复杂程度
  • 增加新角色
  • 增加新的副线
  • 穿墙继续前进

# 删减过于庞大的中段

  • 合并或删掉角色
  • 合并副线情节
  • 削减无聊程度

# 结尾

  • 结局感觉完全符合这本小说所属的类型
  • 结局也让读者感到惊讶

# 冲击结尾
